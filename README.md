# MarketingSelector

Выбор рекламной кампании на основе анализа иерархий (АИИ) и матричных игр.

## Описание

Приложение решает задачу выбора рекламного канала в два этапа:

1. **АИИ** — оценка доступных каналов продвижения по нескольким критериям
   (охват, CAC, релевантность) с учётом весов, заданных пользователем.
2. **Матричная игра** — моделирование реакции конкурента на выбор канала.
   Поиск равновесия Нэша в смешанных стратегиях определяет вероятностное
   распределение бюджета, максимизирующее ожидаемую прибыль.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run ui/app.py
```

├── config/params.yaml    # конфигурация каналов и критериев
├── data/

│   └── payoffs.py        # расчёт платёжных матриц
├── core/
│   ├── ahp.py            # АИИ-ранжирование каналов
│   ├── game.py           # равновесие Нэша
│   └── models.py         # Pydantic-схемы
├── docs/
│   ├── scenarios.md      # демо-сценарии
│   └── coverage/         # отчёт о покрытии тестов
├── ui/
│   ├── app.py            # Streamlit-приложение
│   └── components.py     # UI-компоненты
├── utils/
│   ├── cache.py          # кэширование расчётов
│   └── logger.py         # логирование
├── tests/                # pytest-тесты
├── requirements.txt
└── README.md

## Стек

- **NumPy** — матричные вычисления, собственные числа (AHP), платёжные матрицы
- **Nashpy** — поиск равновесия Нэша в биматричных играх
- **Streamlit** — веб-интерфейс
- **Pydantic** — валидация данных
- **matplotlib** — визуализация (круговые диаграммы)
- **pytest** — тестирование
